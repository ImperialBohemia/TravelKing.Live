# Palette's Journal

## 2024-05-23 - SearchBox Accessibility
**Learning:** Making decorative elements (like swap icons) functional is a huge usability win. Users expect interactivity.
**Action:** Always check if icon-only elements should be interactive buttons.

## 2024-05-23 - Form Accessibility
**Learning:** Linking labels to inputs via `htmlFor` and `id` is fundamental but often missed.
**Action:** Always verify label-input association.
